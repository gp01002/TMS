📖 專案簡介
本系統是專為 艾柏國際芳香療法師（IFA認證） 設計的瑞典式腿部按摩3D互動教學平台。基於2015年IFA認證考試標準手法，結合現代3D可視化技術與Web3.0架構，提供學員與專業按摩師一個創新的學習與展示工具。
🎯 核心特色

✅ 專業認證標準 - 符合IFA國際芳香療法師認證考試要求
✅ 3D互動展示 - 360度旋轉查看，支援多種3D模型格式
✅ 五大手法完整教學 - 推撫、揉捏、摩擦、叩擊、震顫詳細說明
✅ 響應式設計 - 支援桌面、平板、手機等多種裝置
✅ Web3.0準備 - 架構支援未來區塊鏈與NFT整合


🎓 關於 IFA 與瑞典式按摩
國際芳香療法師協會（IFA）
IFA (International Federation of Aromatherapists) 成立於1985年，是全球最具權威性的芳香療法認證機構之一。IFA認證代表：

📜 國際認可的專業資格
🏆 嚴格的教育與考核標準
🌍 全球超過40個國家的專業網絡
💼 保險與專業執業保障

瑞典式按摩（Swedish Massage）
瑞典式按摩由瑞典醫生 Per Henrik Ling 於19世紀初創立，是現代按摩療法的基礎。
五大基本手法
手法英文名稱技巧特點主要功效🖐️ 推撫法Effleurage長而流暢的推動暖和肌肉、均勻塗抹按摩油👐 揉捏法Petrissage捏、提、擠壓或捲動促進血液循環、深層肌肉放鬆👆 摩擦法Friction小幅度圓形或橫向摩擦鬆解肌腱、預防疤痕組織✊ 叩擊法Tapotement快速有節奏地拍打刺激肌肉和神經🌊 震顫法Vibration輕微快速的顫動深度放鬆緊繃區域

📦 支援的3D模型格式
推薦格式
格式副檔名優勢建議使用情境GLB.glb⭐⭐⭐⭐⭐ 最佳選擇單一檔案、包含材質貼圖GLTF.gltf⭐⭐⭐⭐ 推薦可編輯性高、開放標準OBJ.obj⭐⭐⭐ 支援通用性強、需搭配MTL檔FBX.fbx⭐⭐⭐ 支援動畫支援、Autodesk標準
3D模型建議規格
- 多邊形數量：建議 10,000 - 50,000 面
- 材質貼圖：PNG/JPG，建議 2048x2048
- 模型尺寸：建議單位為公尺，腿部長度約 0.8-1.2m
- 原點位置：建議設定在模型中心或底部

❓ 常見問題（FAQ）
Q1: 為什麼我上傳的模型無法顯示？
A: 請確認：

檔案格式是否為 GLB, GLTF, OBJ, FBX
檔案大小是否過大（建議 < 50MB）
模型是否包含有效的幾何資料
瀏覽器是否支援 WebGL

Q2: 可以商業使用嗎？
A: 可以！本專案採用 MIT 授權，允許商業使用。但請注意：

IFA 認證內容需遵守 IFA 規範
3D 模型版權需另行確認
建議保留原始版權聲明

Q3: 如何加入動畫演示？
A: 您需要：

準備包含動畫的 GLB/FBX 檔案
使用 Three.js AnimationMixer 播放動畫
參考 Three.js 官方文件：https://threejs.org/docs/#api/en/animation/AnimationMixer

Q4: 支援手機瀏覽嗎？
A: 完全支援！本系統採用響應式設計：

📱 手機：觸控拖曳旋轉模型
📲 平板：完整功能體驗
💻 桌面：最佳視覺效果

Q5: 如何整合到現有網站？
A: 兩種方式：

iframe 嵌入：<iframe src="path/to/html" width="100%" height="800"></iframe>
直接整合：將 HTML 內容拆分為 CSS/JS 檔案後引入
