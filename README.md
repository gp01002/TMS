瑞典式腿部按摩標準流程指引
專案簡介 (Introduction)
本軟體旨在協助芳香療法學習者與專業人員，掌握 艾柏國際芳香療法師（2015年標準） 的瑞典式腿部按摩考試手法。透過數位化的方式，呈現從初始上油到最終安撫的完整標準作業程序，確保使用者能精準執行每一個按摩步驟與穴位操作。

核心內容與特色 (Key Features)
本軟體收錄了基於考試標準的詳細手法拆解，包含以下重點流程：
1. 標準化預備與上油流程
• 初始接觸：由毛巾上俯順至大腿，並由兩側滑下至腳底的起始動作。
• 上油路徑：精確指引由阿基里斯腱（Achilles tendon）開始，雙手同時向上的上油技巧。
• 操作順序：遵循考試標準，先覆蓋右腳，由左腳開始進行操作。
2. 專業腿部按摩手法
本軟體詳細記錄了多種關鍵手法的操作要點：
• 長推與回包：展示雙手拇指並排（原文為病牌）長推至大腿，並由兩側回包的動作軌跡。
• 深層肌肉放鬆：包含「由下至上」的撥提手法，以及針對大腿內外側的「S型抓捏」技巧。
• 流暢連接：演示雙手如何交替向上撫順（輔順），以維持按摩的連貫性。
3. 精細分區與穴位護理
針對小腿與足部提供精確的分區指引：
• 小腿分區：詳解推至委中穴（原文為尾中穴）的路徑，並將小腿由內自外分為三等份（內中外、外中內）進行來回推撫。
• 足部操作：
◦ 包含將腳抬起呈 90 度與托住腳盤的正確姿勢。
◦ 針對湧泉穴（原文為永泉）的順時針畫圈按摩。
◦ 利用半握拳方式，由腳跟往腳趾方向的滑動技巧。
5. 完整的結束流程
• 指引標準的「大安撫」（Big Effleurage）動作。
• 包含最後將雙手放置於腳底停留，並蓋上毛巾的完整收尾步驟。
